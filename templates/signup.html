How to Get Your QRZ API Key

  1. Log in to QRZ.com
  2. Go to My Logbook
  3. Click Settings
  4. Select the API tab
  5. Copy your API Key

Note: A QRZ XML Subscription is required to access the API. Your API key allows us to sync your QSOs from QRZ to track your competition progress.

Your API key is encrypted and stored securely. We only use it to fetch your logbook data.
